DeviceStreamEvent.Peripheral.LocationState
  • LocationState

    Physical location state for an Afero device.

    Afero devices can optionally be given a physical location. This may be assigned by a hub, optionally by the user during initial association, or afterward by the user in device settings.

    Interpretation

    • unknown: The location state has been fetched from the service, and the service has responded that the device does not have a set location.

    • known(location): The device has a known location.
