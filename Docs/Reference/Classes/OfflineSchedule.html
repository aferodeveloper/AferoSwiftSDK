  • FromBytes(_:types:)

    Parse a single ScheduleEvent from a byteArray.

    This method takes an array of bytes, and a map of attributeIds to DataTypes, and attempts to parse a full Schedule event, returning the event and number of bytes consumed. If data are incomplete, the method returns (nil, 0). If various other conditions are not met, the method will throw (see below for details).

    Note

    note:

    • If data are incomplete, we return (nil, 0), since it’s possible we just have an empty chunk.
    • If data are complete, but any of the following conditions are met, then we throw:
    • The TimeSpecification does not parse.
    • The AttributeSpecification does not parse.
    • The AttributeId is not contained in the types map
    • The type mapped to the given AttributeId is not of fixed size.

    Declaration

    Swift

    public static func FromBytes(_ bytes: [UInt8], types: [Int: DeviceProfile.AttributeDescriptor.DataType]) throws -> ScheduleEventFromBytes

    Parameters

    bytes

    A [UInt8] containing the ScheduleEvent data staring at index 0.
    types

    A map of attributeIds to DataTypes. All referenced ids in the serialized data must dereference from this map, otherwise we throw.

    Return Value

    The event, if any, that was parsed out, along with the number of bytes, if any, consumed from the data.
