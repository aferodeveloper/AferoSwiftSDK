  • Rule(_:match:apply:)

    Given a function that produces a value (fetch), a function which interrogates the value and returns a boolean (match), and a function which applies changes to input and returns output, return a function which:

    1. Accepts a context C and a target T
    2. Fetches the the expected value V? by passing C to the fetch function.
    3. Evaluates the resulting V? according to the match
    4. Applies the transformation (T)->T and returns the transformed value if the match in (3) was positive, or simply returns the unchanged input value if the match was negative.

    Swift

    open class func Rule<C, V, T>(
        _ fetch: @escaping ((C)->V?),
        match: @escaping (V?)->Bool,
        apply: @escaping (T)->T
        ) -> ((C?, T)->T)

    Parameters

    match

    apply
